# NightBite

NightBite es un videojuego móvil 2D de terror psicológico y comedia desarrollado en Android Studio para la clase de POO II.

El jugador toma el papel de un repartidor nocturno atrapado en una dimensión paranormal después de aceptar un pedido extraño dentro de una aplicación de delivery. A medida que completa entregas y explora escenarios oscuros, descubrirá secretos ocultos, clientes perturbadores y anomalías cada vez más peligrosas.

El proyecto utiliza:
- Kotlin
- LibGDX
- Tiled Map Editor
- Programación Orientada a Objetos (POO)

## Género
Terror psicológico • Exploración • Narrativa interactiva

## Plataforma
Android
